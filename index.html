Leia sobre as tecnologias integradas

Jekyll

Jekyll é um gerador de sites estáticos. Ele pega o texto escrito em sua linguagem de marcação favorita e usa layouts para criar um site estático.

Tailwind CSS

Tailwind CSS é um framework utility-first para construir interfaces customizadas rapidamente, sem sair do HTML.

API's

Integração completa com API's externas e gratuitas para geração de comentarios, via Disqus, contadores de visualização e mais!

Github Pages

As páginas do GitHub são páginas da web públicas hospedadas gratuitamente no github.io ou em um domínio personalizado de sua escolha.
